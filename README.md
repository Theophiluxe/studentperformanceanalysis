# TECH STUDIO STUDENT PERFORMANCE ANALYSIS
This project analyzes student enrollment, academic performance, revenue trends, and instructor workload for a multi-program technology academy using Power BI. The goal was to identify enrollment imbalances, performance gaps across courses and study modes, and assess whether revenue and enrollment targets were being achieved.

Project Overview

This project presents a comprehensive business and academic performance analysis for Tech Studio Academy, a technology training institute offering programs in Data Science, Web Development, Cybersecurity, UI/UX Design, Artificial Intelligence, and Digital Marketing.
The objective was to transform raw institutional data into actionable insights that support decision-making around enrollment growth, student success, operational efficiency, and revenue performance.

Business Problem

The academy observed several operational challenges:
Uneven enrollment across courses
High dropout rates in specific programs
Differences in student performance by study mode
Revenue below institutional targets
Imbalanced instructor workload distribution

This analysis was conducted to identify root causes and recommend data-driven improvements. 

Tools & Technologies

Power BI — Data modeling, dashboard development, visualization

DAX (Data Analysis Expressions) — KPI calculations and business metrics

Data Modeling — Relationship building across multiple datasets

EDA Techniques — Data validation and preparation 

Dataset Description

The analysis integrates multiple institutional datasets:

Students Data — Demographics and enrollment information

Performance Data — Academic results and scores

Instructors Data — Teaching assignments and workload

Finance Data — Payments and revenue records

Targets Data — Enrollment and revenue benchmarks 

Key Insights

Artificial Intelligence recorded the highest enrollment, while Digital Marketing had the lowest. 

Dropout rates were highest in Digital Marketing (25%) and Web Development (20%). 

Physical learning students outperformed online students academically (71 vs 64 average score). 

Revenue contribution varied by campus, with the West campus generating the most income. 

The institution shows strong growth potential but remains significantly below enrollment targets. 

Dashboard Features

The Power BI dashboard enables stakeholders to explore:
Enrollment distribution by course and campus
Dropout and retention patterns
Instructor workload analysis
Performance comparison by study mode
Revenue trends and payment channels
Target vs Actual performance tracking
Business Value

This project demonstrates how combining academic, operational, and financial data can:

✔ Identify high-risk programs early
✔ Improve student retention strategies
✔ Optimize instructor allocation
✔ Support data-driven recruitment planning
✔ Enhance institutional revenue performance
